# Trabajo Backend 1 Coderhouse

Proyecto realizado como parte del curso de **Backend I** de Coderhouse.

Este proyecto consiste en una aplicación backend utilizando **Node.js** con **Express**, **MongoDB** y plantillas con **Handlebars**. Se implementaron funcionalidades básicas de manejo de datos y alertas interactivas para el usuario.

## Tecnologías utilizadas

* **Node.js**
* **Express**
* **MongoDB**
* **Mongoose**
* **Handlebars**
* **SweetAlert**
* **dotenv**

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/Gonzanaio/BackEnd-I-Coderhouse_Gonzalo-Naiouf.git
```

2. Instalar dependencias:

```bash
npm install
```

3. Crear un archivo `.env` en la raíz del proyecto con las variables necesarias:

```env
MONGO_URI=tu_url_de_mongodb
PORT=3000
```

## Uso

1. Levantar el servidor:

```bash
npm start
```

2. Abrir en el navegador:

```
http://localhost:3000
```

3. Interactuar con la aplicación y ver alertas con SweetAlert.

## Funcionalidades

* Conexión a **MongoDB** con Mongoose
* Renderizado de vistas con **Handlebars**
* Manejo de rutas con **Express**
* Alertas interactivas con **SweetAlert**
* Configuración de variables de entorno con **dotenv**
